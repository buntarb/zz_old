ordainSanitizedHtml(content, opt_contentDir)

Takes a leap of faith that the provided content is "safe" HTML.

Parameters
content*

A string of HTML that can safely be embedded in a PCDATA context in your app. If you would be surprised to find that an HTML sanitizer produced s (e.g. it runs code or fetches bad URLs) and you wouldn't write a template that produces s on security or privacy grounds, then don't pass s here.

opt_contentDir?number=

The content direction; null if unknown and thus to be estimated when necessary. Default: null.

Returns
soydata.SanitizedHtml

Sanitized content wrapper that indicates to Soy not to escape when printed as HTML.

ordainSanitizedUri(content)

Takes a leap of faith that the provided content is "safe" to use as a URI in a Soy template.

This creates a Soy SanitizedContent object which indicates to Soy there is no need to escape it when printed as a URI (e.g. in an href or src attribute), such as if it's already been encoded or if it's a Javascript: URI.

Parameters
content*

A chunk of URI that the caller knows is safe to emit in a template.

Returns
soydata.SanitizedUri

Sanitized content wrapper that indicates to Soy not to escape or filter when printed in URI context.
